🎼 YTM Immersion v1.0.0 [Stable Beta]
YouTube Music のウェブ版を、Apple Music (Mac/iPad) のような没入感のある歌詞プレイヤーに変える拡張機能です。

あなたが苦労して修正した**「画面遷移バグ」や「ガラス化」**の魅力を伝える内容になっています。

✨ 拡張機能の主な特徴 (Features)
完全没入型レイアウト: 画面上の邪魔な要素（サイドバー、余計なメニュー）を消去し、アートワークと歌詞だけに集中できるクリーンなデザイン。

ガラスモーフィズムUI: プレイヤーバーや上部ナビゲーションを半透明で浮遊感のある「リキッドグラス」デザインに統一。

リアルタイム歌詞同期: LrcLibとの連携により、曲に合わせて歌詞が滑らかに追従。

安定動作: 複雑なSPA（画面遷移）にも対応し、ロゴをクリックしてホームに戻ると自動で没入モードが解除されます。

クロスブラウザ対応: ChromeとFirefox（WebExtensions対応ブラウザ）の両方で動作します。

🚀 インストール方法 (Installation)
この拡張機能はストア公開前の「開発者モード」でのみ利用可能です。

【重要】以下の手順を正確に行ってください。

ファイルのダウンロード

このリポジトリの [Releases] ページへ移動し、最新の YTM-Immersion-vX.X.X.zip ファイルをダウンロードします。

ダウンロードしたファイルを解凍（展開）し、拡張機能フォルダを取り出します。

デベロッパーモードの有効化

Google Chrome (または Edge, Brave) を開きます。

アドレスバーに chrome://extensions と入力し、拡張機能管理画面を開きます。

画面右上の 「デベロッパーモード」 (Developer Mode) をONにします。

ファイルの読み込み

画面左上の 「パッケージ化されていない拡張機能を読み込む」 ボタンをクリックします。

手順1で解凍した フォルダ全体 を選択して読み込みます。

完了

読み込みが完了したら、YouTube Musicを開き、曲を再生してください。

🎵 使い方とトラブルシューティング
ON/OFFの切り替え:

画面右下にある [IMMERSION] ボタンをクリックすると、没入モードのON/OFFが切り替わります。

ホーム画面への復帰:

没入モードONのままロゴをクリックしてホームに戻っても、自動的に通常モードに戻り、ボタンが押せるようになります。

歌詞が出ない曲がある:

この拡張機能はLrcLibのデータベースに依存しています。もし歌詞が出ない場合は、まだデータベースに登録されていない曲です。
